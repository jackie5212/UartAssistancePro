# 开发排期（人天估算，React + Tauri 跨平台版）

## 1. 估算前提

- 角色：1 名全栈工程师（React + Rust）
- 目标：10 周完成可商用 Beta（P0 + 核心 P1 + 部分 P2）
- 不含：大规模企业定制、复杂协议库全量覆盖

## 2. 里程碑与人天

### W1-W2（10 人天）跨平台脚手架与 UI 骨架

- Tauri + React + TypeScript 工程初始化
- UI 主框架与状态管理基线
- 主窗口、参数面板、接收区、发送区、状态栏可交互

### W3-W4（10 人天）Rust 串口能力接入

- Rust 串口核心抽象层（扫描、打开、关闭）
- Tauri Command/事件通道打通（前后端联调）
- 基础发送/接收（ASCII/HEX）+ 时间戳/统计/清空/暂停

### W5（5 人天）编码与体验补齐

- 编码自动识别与手动切换
- 配置持久化（最近端口、参数）
- 错误提示统一与异常恢复

### W6-W7（10 人天）P1 高级交互

- 定时/循环/队列发送
- 白黑名单/正则/按帧过滤
- 关键字高亮与异常标记

### W8-W9（10 人天）P2 解析能力

- CRC 与数据转换工具
- Modbus 基础解析
- 协议模板最小可用版本

### W10（5 人天）收尾与三平台发布

- 跨平台联调与回归修复
- Windows/msi、macOS/dmg、Linux/AppImage 产出
- 文档整理与版本发布

## 3. 总计

- **建议团队：2 人（开发 1 + 测试/产品兼职 1）**
