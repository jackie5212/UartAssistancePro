# 跨平台专业串口调试工具 PRD v1.0

## 1. 项目概述

- 产品名：跨平台专业串口调试工具
- 目标平台：Windows / macOS / Linux
- 用户群体：嵌入式工程师、硬件开发、工控从业者、电子专业学生
- 产品版本：免费版 / Pro 版 / 企业版
- 目标：新手可快速上手，专业用户可深度定制

## 1.1 技术路线（已更新）

- 前端：React + TypeScript
- 桌面容器：Tauri 2
- 串口核心：Rust（Tauri Command + Event）
- 构建目标：Windows / macOS / Linux 一套代码发布
- 说明：后续不再以 Win32 单平台作为主线开发

## 2. 版本范围

### 免费版（首发）

- 串口管理：扫描、刷新、参数配置、打开关闭
- 基础收发：ASCII/HEX、时间戳、统计、暂停、清空
- 乱码处理：UTF-8/GBK 自动识别与手动切换
- 配置持久化：最近端口与参数自动保存
- 基础日志：TXT/CSV 导出

### Pro 版

- 高级发送：定时/循环/序列/触发
- 高级接收：白黑名单、正则、按帧过滤、高亮
- 校验与转换：CRC/进制/大小端/浮点转换
- 协议解析：Modbus + 自定义模板
- 波形可视化：多通道基础波形与导出

### 企业版

- 多串口并发（2-32）
- 自动化脚本（Lua/Python）
- 报警联动、日志加密、审计导出
- 定制协议扩展与私有化部署

## 3. 优先级规划

### P0（必须）

- 串口端口管理（枚举、状态、打开关闭）
- 基础收发与统计
- 时间戳（绝对/相对，ms/us）
- 编码切换（解决乱码）
- 配置持久化
- 三平台可运行

### P1（高优先）

- 定时/循环/队列发送
- 触发发送
- 过滤与高亮
- 分片发送

### P2（中优先）

- CRC/校验工具
- Modbus 解析
- 自定义协议模板
- 波形展示与导出

### P3/P4（后续）

- 脚本自动化、多串口联动
- 网络透传、蓝牙串口、虚拟串口、AI 辅助

## 4. 首发验收标准（P0）

- Win/macOS/Linux 串口可扫描与打开
- 连续收发 30 分钟无崩溃
- 115200~921600 波特率稳定收发
- 乱码可通过自动或手动编码切换恢复
- 配置重启可恢复
- 异常提示可读（占用、断开、权限不足）

## 5. 风险与对策

- 平台差异：建立平台适配层与统一设备模型
- 性能瓶颈：收发线程与 UI 线程解耦，批量刷新
- 范围膨胀：先聚焦 P0/P1，协议先做 Modbus

## 6. 迁移原则（从当前原型到跨平台）

- 保留现有功能点 ID，不改需求语义
- UI 在 React 重建，Win32 原型仅作交互参考
- 串口逻辑统一下沉到 Rust 层，避免平台分叉
- 每完成一个功能点，同时补前后端联调与错误码
